# Interactive Playground

Enable users to interact with your API and make test requests.

---

## GET /notarization/{id}

The following endpoint retrieves the details of a specific notarization case.

### Authorization
| Name            | Type    | Required | Description                |
|------------------|---------|----------|----------------------------|
| Authorization    | string  | required | Bearer token for access.   |

### Path Parameters
| Parameter | Type   | Required | Description              |
|-----------|--------|----------|--------------------------|
| `id`      | string | required | The unique ID of the notarization. |

---

## Example cURL Request
```bash
curl --request GET \
  --url https://api.platus.com/notarization/{id} \
  --header 'Authorization: Bearer <token>'
