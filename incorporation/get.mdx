---
title: "Get Incorporation"
api: "GET https://api.beglaubigt.de/v1/incorporation/{case_id}"
description: "This endpoint retrieves the details of an incorporation"
authMethod: "bearer"
---

### Authorization

To authenticate requests, include an Authorization header with a Bearer token:

`Authorization: Bearer <token>`

**Example:** `Bearer sk-123e4567-e89b-12d3-a456-426614174000`

### Path Parameters

<ParamField path="incorporation_id" type="string" required>
  The unique identifier of the incorporation to retrieve.

  **Example**: `123e4567-e89b-12d3-a456-426614174000`
</ParamField>

### Response

<ResponseField name="result" type="string">
  Indicates the outcome of the API request.

  **Possible values**: `success` or `error`
</ResponseField>

<ResponseField name="message" type="string">
  Provides a human-readable message detailing the result of the API request.

  **Exemple**: `Incorporation fetched successfully.`
</ResponseField>

<ResponseField name="incorporation_id" type="string">
  The unique identifier for the incorporation in a **UUID** format.

  **Example**: `123e4567-e89b-12d3-a456-426614174000`
</ResponseField>

<ResponseField name="name" type="string">
  The legal name of the company.

  **Example**: `Acme Holdings GmbH`
</ResponseField>

<ResponseField name="type" type="string">
  The type of company incorporated.

  **Possible values**: `GmbH` or `UG`
</ResponseField>

<ResponseField name="purpose" type="string">
  The purpose or business objective of the company.

  **Example**: `Digital Notarization Services for Germany and Europe`
</ResponseField>

<ResponseField name="capital" type="number">
  The total share capital of the company in EUR.

  **Example**: `25000`
</ResponseField>

<ResponseField name="address" type="object">
  The address of the company.

  <Expandable title="Address">
    <ResponseField name="street" type="string">
      The street name and number of the company.

      **Example**: `Schillerstraße 5`
    </ResponseField>
    <ResponseField name="street_second_line" type="string">
      Additional address information, such as apartment number, suite, floor, or building.

      **Example**: `Apartment 4B`
    </ResponseField>
    <ResponseField name="zip" type="string">
      The postal code of the company.

      **Example**: `80336`
    </ResponseField>
    <ResponseField name="city" type="string">
      The city of the company.

      **Example**: `Munich`
    </ResponseField>
    <ResponseField name="country" type="string">
      The country of the company.

      **Example**: `Germany`
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="status" type="string">
  The status of the incorporation.

  **Example**: `Initiated` or `Completed`
</ResponseField>

<ResponseField name="url" type="string">
  The URL of the incorporation.

  **Example**: `https://app.beglaubigt.de/incorporation/123e4567-e89b-12d3-a456-426614174000`
</ResponseField>

<ResponseField name="directors" type="array">
  An array of directors managing the company.

  <Expandable title="Director">
    <ResponseField name="first_name" type="string">
      First name of the director.\
      **Example**: `Maria`
    </ResponseField>
    <ResponseField name="last_name" type="string">
      Last name of the director.\
      **Example**: `Weber`
    </ResponseField>
    <ResponseField name="email" type="string">
      Email address of the director.\
      **Example**: `maria.weber@example.com`
    </ResponseField>
    <ResponseField name="phone" type="string">
      The phone number of the director.\
      **Example**: `+491234567890`
    </ResponseField>
    <ResponseField name="dob" type="string">
      Date of birth of the director in `YYYY-MM-DD` format.\
      **Example**: `1978-03-15`
    </ResponseField>
    <ResponseField name="address" type="object">
      Address of the director.
      <Expandable title="Address">
        <ResponseField name="street" type="string">
          The street name and number of the director.\
          **Example**: `Schillerstraße 5`
        </ResponseField>
        <ResponseField name="street_second_line" type="string">
          Additional address information, such as apartment number, suite, floor, or building.\
          **Example**: `Apartment 4B`
        </ResponseField>
        <ResponseField name="zip" type="string">
          The postal code of the director.\
          **Example**: `80336`
        </ResponseField>
        <ResponseField name="city" type="string">
          The city of the director.\
          **Example**: `Munich`
        </ResponseField>
        <ResponseField name="country" type="string">
          The country of the director.\
          **Example**: `Germany`
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="authority" type="string">
      The director’s signing authority.\
      **Possible values**: `sole` or `joint`
    </ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="documents" type="array">
  A list of documents included in the incorporation.

  <Expandable title="Document">
    <ResponseField name="name" type="string">
      The name of the document.
      **Example**: `document.pdf`
    </ResponseField>
    <ResponseField name="type" type="string">
      The type of document.
      **Possible values**: `standard` or `custom`
    </ResponseField>
    <ResponseField name="status" type="string">
      The status of the document.
      **Possible values**: `Initiated` or `Completed`
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created_at" type="string">
    The timestamp at which the incorporation was created, formatted as an ISO 8601 string (UTC).\
  **Example**: `2025-06-15T08:00:00.000Z`
</ResponseField>

<ResponseField name="client_reference" type="string">
  A custom client reference string for tracking purposes.\
  **Example**: `REF-2025-001`
</ResponseField>
