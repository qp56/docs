# Create Notarization

This endpoint allows you to create a notarization request for a document. It includes the jurisdiction, the document to be notarized, and the signer details.

---

## Body

<ParamField name="jurisdiction" type="string" required>
The jurisdiction in which the notarization will be executed. Example: `"US"`.
</ParamField>

<ParamField name="state" type="string" required>
The state or region within the jurisdiction. Example: `"CA"`.
</ParamField>

<ParamField name="documents" type="array" required>
An array of documents to be notarized. Each document should include `document_type`, `document_file`, `is_already_signed`, and `signers`.
</ParamField>

<Expandable title="Document Object">
  <ParamField name="document_type" type="string" required>
  The type of document. Example: `"Power of Attorney"`.
  </ParamField>
  <ParamField name="document_file" type="object" required>
  File details for the document, including `type` and `url`.
  </ParamField>
  <ParamField name="is_already_signed" type="boolean">
  Whether the document is already signed before notarization.
  </ParamField>
  <ParamField name="signers" type="array" required>
  An array of signer objects containing `email`, `first_name`, `last_name`, `date_of_birth`, and more.
  </ParamField>

  <Expandable title="Signer Object">
    <ParamField name="email" type="string" required>
    Email address of the signer.
    </ParamField>
    <ParamField name="first_name" type="string" required>
    First name of the signer.
    </ParamField>
    <ParamField name="last_name" type="string" required>
    Last name of the signer.
    </ParamField>
    <ParamField name="date_of_birth" type="string">
    Date of birth of the signer in `YYYY-MM-DD` format.
    </ParamField>
    <ParamField name="labels" type="array">
    Label information for signature positioning or other metadata.
    </ParamField>
  </Expandable>
</Expandable>

<ParamField name="client_reference" type="string">
A custom client reference string for tracking purposes. Example: `"REF-2024-001"`.
</ParamField>

<ParamField name="client_postback_url" type="string">
A URL where status updates will be sent. Example: `"https://client-domain.com/webhook-endpoint"`.
</ParamField>

---

## Response

<ResponseField name="case_id" type="string">
The unique identifier for the created notarization case.
</ResponseField>

<ResponseField name="status" type="string">
The status of the notarization request. Example: `"created"`.
</ResponseField>

<ResponseField name="documents" type="array">
An array of documents with metadata about their processing state.
</ResponseField>

---
