# Initiate Incorporation

**API**: `POST https://api.platus.com/v1/incorporation`  
**Description**: This endpoint creates an incorporation request for a company, such as a GmbH or UG, including company details, shareholders, and directors.

## Authorization

<ParamField header="Authorization" type="string" required>  
Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.  
**Example**: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`  
</ParamField>

## Body

<ParamField body="company" type="object" required>  
Details of the company being incorporated.  
  <Expandable title="Company Object">  
    <ParamField body="name" type="string" required>  
    The legal name of the company.  
    **Example**: `Platus GmbH`  
    </ParamField>  
    <ParamField body="type" type="string" required>  
    The type of company being incorporated.  
    **Possible values**: `GmbH`, `UG`  
    **Example**: `GmbH`  
    </ParamField>  
    <ParamField body="purpose" type="string" required>  
    The purpose or business objective of the company.  
    **Example**: `Digital Notarization Services`  
    </ParamField>  
    <ParamField body="address" type="object" required>  
    The registered address of the company.  
      <Expandable title="Address Object">  
        <ParamField body="street" type="string" required>  
        The street name and number.  
        **Example**: `Platusstraße 1`  
        </ParamField>  
        <ParamField body="postal_code" type="string" required>  
        The postal code.  
        **Example**: `80331`  
        </ParamField>  
        <ParamField body="city" type="string" required>  
        The city.  
        **Example**: `Munich`  
        </ParamField>  
        <ParamField body="country" type="string" required>  
        The country.  
        **Example**: `Germany`  
        </ParamField>  
      </Expandable>  
    </ParamField>  
    <ParamField body="capital" type="number" required>  
    The total share capital of the company in EUR.  
    **Example**: `25000`  
    </ParamField>  
    <ParamField body="shares" type="array" required>  
    An array of share allocations for the shareholders.  
      <Expandable title="Share Object">  
        <ParamField body="shareholder_id" type="number" required>  
        The unique identifier linking to a shareholder in the `shareholders` array.  
        **Example**: `0`  
        </ParamField>  
        <ParamField body="share_percentage" type="number" required>  
        The percentage of total shares owned by the shareholder.  
        **Example**: `50`  
        </ParamField>  
        <ParamField body="share" type="number" required>  
        The monetary value of the shares owned by the shareholder in EUR.  
        **Example**: `12500`  
        </ParamField>  
      </Expandable>  
    </ParamField>  
  </Expandable>  
</ParamField>

        The street name and number.  
        **Example**: `Bahnhofstraße 10`  
        </ParamField>  
        <ParamField body="postal_code" type="string" required>  
        The postal code.  
        **Example**: `10115`

        <ParamField body="city" type="string" required>  
        The city.

## Response

<ResponseField name="result" type="string">  
Indicates the outcome of the API request.  
**Possible values**: `success`, `error`  
**Example**: `success`  
</ResponseField>  

<ResponseField name="message" type="string">  
Provides a human-readable message detailing the result of the API request.  
**Example**: `Case created successfully.`  
</ResponseField>  

<ResponseField name="case_id" type="string">  
The unique identifier for the created incorporation case in a UUID format.  
**Example**: `450506c1-d1a7-46e2-aca6-41a816805595`  
</ResponseField>  

<ResponseField name="status" type="string">  
The current status of the incorporation request.  
**Possible values**: `Incorporation Initiated`, `In Progress`, `Completed`  
**Example**: `Incorporation Initiated`  
</ResponseField>  

<ResponseField name="url" type="string">  
A unique, case-specific URL for accessing and monitoring the incorporation process.  
**Example**: `https://app.platus.com/incorporation/450506c1-d1a7-46e2-aca6-41a816805595`  
</ResponseField>  

<ResponseField name="documents" type="array">  
List of documents generated for this incorporation request.  
  <Expandable title="Documents Response Object">  
    <ResponseField name="id" type="string">  
    The unique identifier for the document in a UUID format.

    </ResponseField>  
    <ResponseField name="type" type="string">  
    The type of document.  
    **Example**: `Articles of Association`  
    </ResponseField>  
    <ResponseField name="status" type="string">  
    Indicates the current processing state of the document.  
    **Possible values**: `Initiated`, `In Draft`, `Completed`  
    **Example**: `Initiated`  
    </ResponseField>  
    <ResponseField name="url" type="string">  
    URL to access the document.  
    **Example**: `https://app.platus.com/documents/450506c1-d1a7-46e2-aca6-41a816805595`  
    </ResponseField>  
  </Expandable>  
</ResponseField>

Details of the notarization process associated with the incorporation.
